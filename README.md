# Vention - 발명 생산성 웹앱

중고등학생을 위한 AI 기반 발명 아이디어 개발 플랫폼

## 주요 기능

- 📋 **AI 변리사**: 특허 검색 및 상담
- 🤓 **아이디어 챗봇**: SCAMPER 기법 기반 아이디어 발전
- ✏️ **AI 스케치 정교화**: 손그림을 AI로 개선
- 🎨 **3D 모델링**: 아이디어 3D 시각화
- 👁️ **3D 뷰어**: 3D 모델 확인
- 👨‍🏫 **교사 모니터링**: 학생 활동 관리 대시보드

## 시작하기

1. Google 계정으로 로그인
2. 발명 아이디어 입력
3. AI 챗봇과 대화하며 아이디어 발전

## 기술 스택

- React + TypeScript + Vite
- Firebase Authentication
- Google Generative AI (Gemini)
- Netlify 배포
